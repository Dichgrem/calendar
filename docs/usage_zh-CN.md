# 使用指南

## 首次使用

1. 访问应用，首次使用会显示注册页面
2. 设置用户名和密码
3. 登录后进入日历视图

## 深色模式

将鼠标悬停在右下角 + 按钮上，会弹出深色模式切换按钮。点击月亮图标切换深色，太阳图标切换浅色。偏好保存在浏览器中。

## 日历视图

- **月份切换**：点击左上角日期，使用年份选择器
- **今天**：点击圆形按钮回到当天
- **创建事件**：点击右下角 + 按钮
- **编辑事件**：点击已有事件
- **搜索**：将鼠标悬停在右上角搜索图标上展开搜索框，支持全局搜索及日历筛选

## 农历显示

在设置中开启"显示农历"，日历格子左侧会显示农历日期。

## ICS 导入

1. 点击设置页面的"导入 ICS 日历"
2. 选择文件导入或输入远程 URL
3. 预览并选择要导入的事件

## ICS 导出

1. 点击设置页面的"导出 ICS 日历"
2. 选择要导出的日历
3. 下载 ICS 文件

## 常用日历订阅

在设置页面点击"导入常用日历"，可一键订阅：
- 中国节假日
- 节日纪念日
- 二十四节气

## 备份与恢复

- **备份数据库**：在设置页面点击"备份数据库"，完成后点击文件名下载 SQLite 文件
- **导出配置**：点击"导出配置"下载 `config.json`，包含你的设置默认值

> 注意：备份仅支持 Docker/Node.js 环境，Cloudflare Workers 不支持。
