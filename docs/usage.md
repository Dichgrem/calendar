# 用户指南

## 快速开始

1. 浏览器打开 `http://localhost:3000`
2. 首次访问注册账户（单用户系统）
3. 自动创建"默认日历"
4. 开始添加事件

## 登录

### 首次访问

显示注册表单，填写用户名和密码后自动登录并跳转到日历页面。

### 再次访问

显示登录表单，输入已注册的用户名和密码。

### 登出

点击导航栏登出按钮，清除会话 cookie，需重新登录。

## 日历视图

### 导航

- 点击日期数字可在当天创建事件
- 顶部 `<` `>` 按钮切换月份
- "今天"按钮跳转到当前日期

### 事件显示

事件以彩色方块显示在对应日期上，颜色与所属日历一致。点击事件可编辑。

### 显示/隐藏日历

在侧边栏点击日历名称切换显示，隐藏日历时其事件不显示。

## 管理日历

### 创建日历

点击日历管理按钮，输入名称和颜色，点"创建"。

### 编辑/删除

点击日历旁的编辑按钮修改名称/颜色，点删除按钮删除日历及其所有事件。

### 排序

在侧边栏拖拽日历调整顺序。

## 管理事件

### 创建事件

1. 点击日历中的某一天
2. 填写标题（必填）、开始/结束时间（必填）
3. 可选：全天、描述、地点、颜色、重复规则（如 `FREQ=WEEKLY`）
4. 点击"保存"

### 编辑/删除

点击事件打开编辑器，修改后保存。删除为软删除——事件仍可通过直接链接访问。

### 重复事件例外

对重复事件的特定日期创建例外：打开事件 → 选择"例外" → 修改标题/时间或标记取消。

## ICS 导入

### 从文件

进入导入页面 → "文件"标签 → 选择 `.ics` 文件 → 勾选要导入的事件 → 导入。

### 从 URL

进入导入页面 → "URL"标签 → 输入订阅地址 → 抓取 → 导入。

### 常用日历

从日历管理面板一键导入预配置的公共日历（法定节假日等）。

## ICS 导出

日历管理面板点击导出图标，自动下载 ICS 文件。

## 设置

| 设置项 | 说明 |
|---|---|
| 语言 | 中文 / English |
| 每周起始日 | 周日(0) 或 周一(1–6) |
| 显示事件时间 | 日历网格中显示具体时间 |
| 日期格式 | zh / en |
| 农历 | 在公历旁显示农历日期 |
| 修改用户名 | 新用户名需唯一 |
| 修改密码 | 输入旧密码和新密码 |

## 备份

进入设置 → "备份数据库" 创建快照。点击文件名下载。
恢复：停止服务 → 用备份文件替换 `data/calendar.db` → 重启。

## 移动端（Android）

1. 打开应用
2. 输入服务端地址（如 `https://calendar.example.com`）
3. 登录

显示与 Web 端一致的日历界面，变更通过 REST API 同步。

## 常见问题

| 问题 | 解决 |
|---|---|
| 登录失败 | 检查密码，或使用修改密码功能 |
| 日历列表为空 | 先创建日历或检查显示切换 |
| ICS 导入 0 个事件 | 文件可能不含 VEVENT，或格式不标准 |
| 日志显示 Database error | 检查磁盘空间和 db 目录权限 |
| 服务无法启动 | 检查 PORT 占用和 DATABASE_URL 目录权限 |
